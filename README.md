# Sales Performance & Forecasting Dashboard

## Project Description
This repository features an end-to-end sales analysis project designed to transform raw transactional data into actionable business intelligence. Developed using Excel and Power BI, this dynamic dashboard provides stakeholders with deep visibility into revenue performance, product popularity, and near-term market trends. By blending descriptive history with forward-looking predictions, the project helps stakeholders identify operational bottlenecks and capitalize on revenue opportunities.

## Key Highlights & Features
*   **Core Financial Metrics:** Engineered comprehensive high-level KPIs to track financial health at a glance, focusing on total Net Sales and ultimate Net Profit.
*   **Product Performance Tracking:** Isolated "Hot Selling Items" based on order volume and total revenue generation to optimize inventory management.
*   **Predictive Analytics:** Implemented time-series forecasting within Power BI to project expected sales volumes and revenue for the upcoming 15 days, aiding in near-term demand planning.
*   **Interactive Drill-Downs:** Designed responsive visuals that allow users to filter data across regions, time periods, and product categories seamlessly.

## Tools & Technologies Used
*   **Microsoft Excel:** Utilized for initial data ingestion, cleaning, handling missing values, and validating data structure.
*   **Power BI (Power Query & DAX):** 
    *   Used **Power Query** to transform, shape, and load the cleaned dataset.
    *   Wrote custom **DAX (Data Analysis Expressions)** formulas to calculate complex KPIs like Net Sales and Net Profit margins.
    *   Utilized Power BI's built-in **Analytics and Forecasting** tools to generate the 15-day forward-looking projection line.
